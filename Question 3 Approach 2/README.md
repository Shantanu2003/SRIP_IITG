# Summary

All the moduled required for this project can be intalled using these commands(for windows only)

## numpy
pip install numpy

## tax
pip install jax

## torch
pip install torch

## torchvision
pip install torchvision

## matplotlib
pip install matplotlib

## torch.nn
pip install nn-torch

## torch optim
pip install torch-optim-sparse

## tqdm
pip install tqdm
